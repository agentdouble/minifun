# Vecteur Zero - FPS LAN

Un FPS multijoueur jouable depuis un navigateur. Le serveur tourne en local et les autres joueurs se connectent via l'IP du reseau local.

## Installation

```bash
npm install
npm start
```

Ouvre ensuite `http://<ip-de-la-machine>:3000` sur chaque ordinateur du reseau local.

## Commandes

- ZQSD / WASD : se deplacer
- Souris : viser
- Clic gauche : tirer
- Clic droit (sniper) : viser au scope
- Tab : afficher le leaderboard
- Shift : sprint
- Espace : sauter
- 1 / 2 / 3 / 4 / 5 : changer d'arme
- R : recharger
- Echap : liberer la souris

## Armes

- Pistolet : cadence moyenne, precis
- Deagle : headshot one-shot
- Fusil : automatique, fiable a moyenne distance
- Sniper : longue portee avec viseur
- Pompe : devastateur a courte portee
